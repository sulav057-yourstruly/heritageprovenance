# Provenance Frontend

React frontend for digital heritage provenance tracking.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Environment

Make sure the backend API is running on `http://localhost:8000` (or set `VITE_API_URL` environment variable).
